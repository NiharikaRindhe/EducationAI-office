# EducationAI-office

React + TypeScript + Vite project for Education AI.

## Technical Stack
- **Frontend**: React, TypeScript, Vite, TailwindCSS / Vanilla CSS
- **Backend**: Node.js, Express, TypeScript (under `/api`)
- **Database**: Supabase / PostgreSQL (migrations under `/supabase`)

## Project Structure
- `/src` - React frontend application source code
- `/api` - Backend Express API source code
- `/supabase` - Database migrations and configurations
